# Iteration Archives

这个目录用于保存每次版本迭代的本地归档，方便和 GitHub 远端仓库形成双重备份。

## 命名规则

- 目录格式：`YYYY-MM-DD-topic-vN`
- 示例：`2026-06-10-sufe-ai-resume-v1`

## 使用规则

1. 每次重要迭代都新增或更新一个日期目录。
2. 目录内至少包含一份可复用文档，比如 PRD、调研、方案或复盘。
3. 本地归档完成后，同步提交并推送到 GitHub 仓库，形成第二份备份。
