# Upload Parser Iteration

## 背景

这一轮迭代补齐了 AI 简历助手最基础的文件导入能力，避免用户必须先手动复制 PDF 或 Word 内容才能继续使用润色和 JD 匹配。

## 本次新增

- 首页新增简历文件上传入口
- 支持 `PDF / DOC / DOCX` 三种格式
- 服务端自动抽取文本并回填到“简历内容”文本框
- 保留原有 AI 润色和 JD 匹配流程

## 当前边界

- 适合可选中文本的 PDF 与常见 Word 文档
- 不包含 OCR，所以扫描版 PDF 的提取效果有限
- 目前是“抽文本”能力，还不是“结构化解析成教育/实习/项目字段”

## 验证结果

- `DOCX` 上传解析成功
- `DOC` 上传解析成功
- `PDF` 上传解析成功
- `npm run lint` 通过
- `npm run build` 通过
